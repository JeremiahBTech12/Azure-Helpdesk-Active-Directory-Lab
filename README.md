# Azure-Helpdesk-Active-Directory-Lab
Azure-hosted Help Desk lab simulating an on-prem Active Directory environment with Microsoft 365 administration and ServiceNow ticket workflows.
Objective: Build a cloud-hosted Active Directory Environment to simulate real-world Help Desk scenarios, including user management, Microsoft365 administration, and ServiceNow ticket resolution workflows.

Technologies
-Microsoft Azure
-Windows Server 2022 (AD DS + DNS)
-Windows 10/11
-Microsoft 365 (Developer Tenant)
-ServiceNow (Developer Insatnce)
